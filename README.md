# Overview

This project focuses on analyzing social media advertising campaign data to evaluate campaign performances, audience engagement, and ROI. The Facebook/Instagram Ads dataset was first cleaned using Python and then visualized using Google Looker Studio. The objective of this project is to provide an interactive dashboard that helps businesses understand which ads/campaigns perform best and where improvements can be made for future campaigns.

# Features

Cleans the data.
Calculates key performance metrics such as CTR, conversions, spend, and ROI.
Identifies top-performing ads and campaigns based on engagement and conversions.
Analyzes audience behavior across fields such as age and gender.
An interactive and user-friendly dashboard built in Google Looker Studio.

# Dashboard Features

KPI cards (for quick display of key business metrics)
Visualzations 
  1. Ad ID vs CTR (with age-wise drill-down)
  2. Amount spent by total conversions (bubble chart)
  3. Campaign-wise clicks and conversions (combo chart)
  4. Campaign-wise ROI summary (donut chart)
Interactivity (Dropdown Filters for age, gender and campaign id)
Actionable Reccomendations


# How to Use this Project

1. Run the Python script to clean and preprocess the raw CSV data.
2. Review the cleaned dataset and calculated metrics.
3. Open the Google Looker Studio report.
4. Use interactive filters and drill-downs to explore campaign performance.
5. Analyze ROI and engagement trends to support data-driven marketing decisions.
